# Deploy to Vercel — my.meowchat.store

## Quick Deploy (2 minutes)

1. Go to https://vercel.com/new
2. Import: gotjisamyo/meowchat-merchant-dashboard
3. Settings:
   - Framework: Vite
   - Build Command: npm run build
   - Output Directory: dist
4. Environment Variables:
   - VITE_API_URL = https://api.meowchat.store
5. Click Deploy
6. After deploy: Settings → Domains → Add → my.meowchat.store
